### Install the necessary packages

```
npm install
# or
yarn install
```

### Replace .env and .env.local GROQ_API_KEY with the your key

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
